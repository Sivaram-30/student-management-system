🔌 REST API Reference

Base URL: http://localhost:8080/api    All protected endpoints require: Authorization: Bearer <JWT>

🔐 Authentication

POST/auth/loginLogin — returns JWT token
POST/auth/registerRegister new user

👨‍🎓 Students (requires ADMIN or TEACHER)

GET/students?page=0&size=10&sortBy=id&direction=ascGet all (paginated)
GET/students/{id}Get by ID
GET/students/search?keyword=aliceSearch students
GET/students/department/{dept}Filter by department
GET/students/course/{course}Filter by course
POST/studentsCreate student (ADMIN only)
PUT/students/{id}Update student (ADMIN only)
DEL/students/{id}Delete student (ADMIN only)
GET/students/statsGet statistics (ADMIN only)
